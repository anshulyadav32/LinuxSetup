Quick Installation

1. Clone the repository

git clone https://github.com/anshulyadav32/LinuxSetup.git
cd LinuxSetup

2. Make scripts executable

chmod +x setup.sh modules/*.sh utils.sh

3. Run the setup

# Install everything
./setup.sh all

# Install specific module
./setup.sh [module-name]

# Test installations
./setup.sh test
